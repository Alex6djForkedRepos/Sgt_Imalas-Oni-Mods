[h1]Improved Radbolt Handling[/h1]

This mod improves the storage and handling of Radbolts.
It allows to change the output direction of the radbolt chamber, gives its logic output reservoir logic and allows setting radbolt directions via ribbon cable.

[h1]Current latest Version: 2.0.0[/h1]
[img]https://raw.githubusercontent.com/Sgt-Imalas/Sgt_Imalas-Oni-Mods/refs/heads/master/Compat_NoBase.png[/img]

[hr][/hr]
[h1]Features[/h1]
[list]
[*] Selectable output direction on the radbolt chamber
[*] Custom anim for the different directions
[*] Mod can be safely removed, all radbolt chambers will revert to facing 
[*] Logic output of the radbolt chamber now behaves like the output of a reservoir
[*] Direction of radbolt output on all radbolt buildings can now be controlled via ribbon input
[/list]
[hr][/hr]
[h1]Known Side Effects:[/h1]
[list]
[*] the output tile indicator is shiftet to occupy the same tile as the input, this is purely visual
[/list]

Note: this mod is a successor to [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2886144715]Rotatable Radbolt Chamber Output[/url] and includes all of its features.
