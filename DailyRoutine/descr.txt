[h1]Introduction[/h1]

This mod adds the option to set daily production routines in production buildings.


[h1]Current latest Version: 1.0.4[/h1]

[hr][/hr]
[h1]Features[/h1]
[list]
[*] Enable repeatable recipes on any building with a complex fabricator. To do so, queue the amount of products you want for each recipe, then click "Enable Daily Routine".
[*] By default the recipe reset happens at the start of a new cycle. This can be adjusted by enabling "Custom Time".
[*] Toggle Recipe Queueing: switch between setting recipe count to a value & Adding the amount to existing number.
[/list]
