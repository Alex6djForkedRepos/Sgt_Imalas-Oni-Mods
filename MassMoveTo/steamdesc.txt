[h1]Mass Move Tool[/h1]

This mod adds a new tool that allows you to select multiple items for a single "move to" command.

[img]https://raw.githubusercontent.com/Sgt-Imalas/Sgt_Imalas-Oni-Mods/refs/heads/master/Compat_All.png[/img]

[h1]Current latest Version: 2.1.4[/h1]

[hr][/hr]

[h1]Features[/h1]
[list][*] Allows you to select an area, then a point to move all items in the area to
[*] Item types have a filtered selection so you can move only specific items
[*] default hotkey: shift+k (can be adjusted in the game settings)
[*] relocate targets are modified to allow multiple dupes delivering to it and fixing the bug that the first item defines the skill required for all further items (allowing critter relocation without ranching). This can be turned off in the config and will also disable itself when the chain tool mod is detected (due to being incompatible with it).
[*] target areas can be drawn over multiple tiles so more deliveries can happen in parallel
[/list]
