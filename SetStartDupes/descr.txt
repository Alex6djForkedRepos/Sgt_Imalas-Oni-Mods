[h1]Introduction[/h1]

This mod allows to adjust the amount of Duplicants you start the game with.

[h1]Current latest Version: 1.0.0[/h1]

[hr][/hr]
[h1]Features[/h1]
[list]
[*] By Default, enabling the mod sets the amount of Duplicants to start with to 1.
[*] This value is adjustable in mod settings to start with up to 6 duplicants.
[/list] 

[h1]Known Issues[/h1]
[list]
[*]Similar to mods done before, having fever than 3 duplicants prevents the Game-Start-Popup to become visible. Hit Esc. to close the invisible startup menu.
[/list]
